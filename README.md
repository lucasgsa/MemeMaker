Esse script cria um meme pre-cadastrado a partir de imagens passadas pelo usuário.
Veja melhor pelo arquivo memes_config.txt.

Para ajudar e criar um template de meme é fácil.
Basta adicionar o templete na pasta Memes, e criar no arquivo de config no formato:
+arquivomeme.jpg
porcentagem_x,porcentagem_y|tamanho_x,tamanho_y
...

é possível adicionar quantas imagens quiser em seu layout.
